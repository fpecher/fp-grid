# fp-grid
This is a simple grid system based ob BEM.

## .container
Simply add <code>.container</code> to your site to wrap everything. If you want to have a fullwidth container use <code>.container--fullwidth</div>.

## .column
Like all the other grid systems simply add <code>.column</code> as the basic class name and then you can extend this class with the class you need. The schema is like: <code>.column--WIDTH</code>.

| column--12   |
| column--11   |
| column--10   |
| column--9    |
| column--8    |
| column--7    |
| column--6    |
| column--5    |
| column--4    |
| column--3    |
| column--2    |
| column--1    |
| column--auto |
